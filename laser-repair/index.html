The homepage should keep its wider public-facing references to industries, supplier ecosystems, supported systems, and products. The directory pages simply add a more disciplined path for buyers who already know the application branch, OEM platform, or equipment model they need help with.

Keep the homepage broad

Structure Rule

How the repair tree is organized.

The public directory does not need to publish all six layers as standalone pages immediately, but the underlying logic is fixed and should stay consistent.

1. Service layer

Laser Repair

2. Application layer

Semiconductor Manufacturing, Display Panel Manufacturing, Lidar, Industrial Processing, Scientific Research

3. OEM / brand

KLA, Applied Materials, Rudolph, LaserScale and similar equipment-side brands.

4. Equipment series

Puma Series, SP Series, Voyager Series and similar family-level groupings when they exist.

5. Equipment model

Puma 9150, Puma 9980, SP2, SP5, UV, Enight and similar model-level entries.

6. Paired laser model

Usually better kept inside the equipment-model page instead of forced into another public page too early.
