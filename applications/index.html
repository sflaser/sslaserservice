How To Choose

Pick the path that matches what you know today.

You do not need perfect information to start. The right entry point depends on whether you know the installed platform, the application goal, or the component need.

You know the installed system

Start with Laser Repair

Use the repair directory when the application area, OEM, or equipment model is already known and the need is review, fault isolation, or repair planning.

You know the target use case

Start with Manufacturing

Use the manufacturing section when the project starts from a performance requirement or system goal rather than an installed failing platform.

You need a part or subassembly

Start with Components

Use the components section when the commercial need is sourcing, replacement, or compatibility review tied to a real platform.
